# Archive: CharCNN (v1.0)
The folder (`CharCNN/`) contains the first phase of the Lipi-Snap project - Character Recognition model using Deep CNN. 

### 📝 Overview
*   **Focus**: Isolated character recognition (Char-by-Char).
*   **Status**: Archived (replaced by the newer Word-level CRNN+CTC system).
*   **Reason for Archive**: To keep the main project clean and organized.

### 📂 What's inside the `CharCNN/` folder?
*   **Scripts**: Original & extend training, evaluation, auditing and inference code.
*   **Tools**: various set of tool for dataset processing and standardisation.
*   **App**: The original Streamlit UI for character recognition.
